# Hi, I'm Enrique 👋

### Linux & Infrastructure Automation Engineer

**Ansible · DevOps · Platform Engineering · Agentic AIOps**

I am a Systems Engineer from the **Universidad Nacional Mayor de San Marcos — UNMSM**, based in Lima, Peru, with **18+ years of experience** in Linux/Unix administration, infrastructure automation, and mission-critical enterprise environments.

I currently work in software engineering and infrastructure automation at **Kyndryl**, developing reusable solutions for Linux RHEL, Windows Server, VMware, Kubernetes, Docker, HAProxy, and AWX/Ansible Automation Platform.

My current professional direction combines **Agentic AI and infrastructure automation**, with a focus on operational knowledge bases, automated remediation, Spec-Driven Development, and reliable deployment workflows.

---

## 🚀 What I work on

* Advanced automation with Ansible roles, collections, inventories, and AWX workflows
* Linux RHEL administration, remediation, compliance validation, and evidence collection
* Infrastructure discovery across VMware, Linux, Windows, AIX, and Power Systems
* Kubernetes deployment, configuration, and platform operations
* Windows Server upgrade automation with PowerShell and VMware Guest Operations
* Python and Bash utilities for JSON, CSV, Excel, validation, and reporting
* CI/CD workflows with GitLab CI, Azure DevOps, Git, and `ansible-lint`
* Agentic AIOps and Spec-Driven Development applied to infrastructure operations
